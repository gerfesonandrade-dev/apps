# 🚀 Ideavault

**Guarde ideias. Transforme em ação.**

O **Ideavault** é um bot inteligente para Telegram que permite salvar, organizar e transformar ideias em informações úteis, como receitas, projetos de construção, decoração, DIY e muito mais.

---

## 💡 Sobre o projeto

O Ideavault funciona como um **cofre de ideias inteligente**, inspirado no Pinterest, mas com foco em produtividade e execução.

Você pode enviar:
- 📝 Texto
- 📸 Imagens
- 🎥 Vídeos

E o sistema:
- organiza automaticamente
- classifica por categoria
- prepara para transformar em algo prático

---

## 🤖 Funcionalidades (MVP)

- Recebe mensagens do Telegram
- Detecta tipo de conteúdo (texto, imagem, vídeo)
- Responde automaticamente ao usuário
- Estrutura pronta para:
  - IA (classificação de ideias)
  - extração de materiais/ingredientes
  - organização por categorias

---

## 🧱 Tecnologias

- Node.js
- Express
- Axios
- Telegram Bot API

---

## ⚙️ Como funciona

1. Usuário envia uma ideia no Telegram
2. O bot recebe via webhook
3. O servidor processa a mensagem
4. O bot responde automaticamente
5. (futuro) IA organiza e salva a ideia
